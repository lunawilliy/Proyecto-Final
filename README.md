# Tolsen React - Proyecto entregable

Instrucciones rápidas:

1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Iniciar en modo desarrollo:
   ```bash
   npm run start
   ```
3. El proyecto usa Vite + React + TypeScript.

Contenido:
- Rutas: /, /destacados, /nuevos, /contacto, /formulario, /resultado
- Contextos: idioma (es/en) y tema (light/dark)
- Banner: transición entre 3 imágenes (CSS)
- Formulario: envía estado a /resultado usando `navigate(..., { state })`

Se entregará además un archivo `link.txt` con la URL del repo cuando subas a GitHub.
